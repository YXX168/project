创建仓库
git init

检查文件状态
git status
git status -s

新文件跟踪和管理
git add 文件名
git add .

提交更新
git commit -m "描述信息"

跳过暂存区提交更新 加-a
git commit -a -m "描述信息"

撤销文件修改，从git仓库还原提交到仓库中的最新文件覆盖本地文件
git checkout -- 文件名

从暂存区中移除
git reset HEAD 要移除的文件名
git reset HEAD .

从仓库和工作区同时移除文件
git rm -f 文件名

从仓库移除此文件，但保留在工作区中
git rm --cached 文件名

创建忽略文件白名单配置文件
.gitignore
#开头是注释
/结尾是目录

查看提交历史
git log
git log -2
git log -2 --pretty=oneline
git log -2 --pretty=format:"%h | %an | %ar | %s"

回退到指定版本
1、先查历史找到指定版本ID
git log
2、回退指定版本
git reset --hard <ID>
3、回退过程中再次查看所有版本信息要用reflog
git reflog --pretty=oneline
4、跳转最新版本
git reset --hard <ID>
